# 奇门遁甲占卜工具

一个基于洛书九宫的奇门遁甲自动排盘系统，集成了排盘、解读、分析与建议功能，用于教学和研究用途。

## 功能特性

### 核心功能

| 功能 | 说明 |
|------|------|
| 自动排盘 | 根据输入时间自动计算四柱干支、阴阳遁、局数 |
| 九宫格显示 | 按洛书方位排列，显示天盘、地盘、人盘、神盘 |
| 所问之事 | 支持输入占卜问题，系统自动识别问题类型 |
| 盘象总结 | 自动总结排盘格局、用神宫位、整体吉凶 |
| 分析建议 | 根据问题类型给出针对性分析和行动建议 |
| 符号解读 | 点击任意符号查看详细的九星、八门、八神解释 |

### 问题类型支持

系统支持以下问题类型的自动识别和分析：

- **事业**：工作、升职、官运
- **财运**：投资、生意、赚钱
- **感情**：婚姻、爱情、对象
- **健康**：疾病、身体
- **学业**：考试、学习
- **出行**：旅游、出差

### 教学功能

- **逐步引导**：从基础概念到排盘方法的完整入门指南
- **案例库**：经典案例分析，涵盖多种实际场景
- **知识图谱**：奇门遁甲核心概念的可视化知识结构

### 界面特性

- **深色主题**：神秘典雅的深色界面设计
- **星空背景**：动态星空动画效果，包含流星特效
- **响应式设计**：适配桌面端和移动端
- **动画效果**：丰富的交互动画，提升用户体验

## 快速开始

### 环境要求

- Node.js 14.0 或更高版本
- npm 6.0 或更高版本

### 安装运行

```bash
# 克隆项目
git clone https://github.com/YOUR_USERNAME/qimen-tool.git
cd qimen-tool

# 安装依赖
npm install

# 启动服务器
npm start
```

打开浏览器访问 `http://localhost:3000`

### 开发模式

```bash
npm run dev
```

使用 `--watch` 参数，文件修改后自动重启服务器。

## 使用说明

### 基本排盘

1. **选择时间**：在页面顶部选择排盘时间（默认为当前时间）
2. **输入问题**（可选）：在"所问之事"输入框中输入占卜问题
3. **点击排盘**：点击"排盘"按钮开始计算
4. **查看结果**：
   - 九宫格显示天盘、地盘、人盘、神盘
   - 元信息面板显示四柱干支和局数
   - 盘象总结显示整体格局
   - 分析建议显示针对性建议

### 查看解读

- **点击宫位**：查看该宫位的详细解读
- **点击符号**：点击九星、八门、八神查看单独解释

### 学习功能

切换标签页查看教学内容：

| 标签页 | 内容 |
|--------|------|
| 逐步引导 | 从基础概念开始学习奇门遁甲 |
| 案例库 | 通过实际案例理解排盘方法 |
| 知识图谱 | 了解奇门遁甲的知识体系 |

## 界面说明

### 九宫格布局

按洛书方位排列：

```
巽4  离9  坤2
震3  中5  兑7
艮8  坎1  乾6
```

每个宫位显示四层信息：

| 层级 | 名称 | 颜色 | 说明 |
|------|------|------|------|
| 神盘 | 八神 | 紫色 | 值符、腾蛇、太阴等 |
| 天盘 | 九星 | 金色 | 天蓬、天芮、天冲等 |
| 人盘 | 八门 | 青色 | 开门、休门、生门等 |
| 地盘 | 九宫 | 灰色 | 坎一宫、坤二宫等 |

### 盘象总结

排盘完成后自动显示：

- **排盘信息**：四柱干支和局数
- **所问之事**：用户输入的问题
- **用神宫位**：根据问题类型自动确定的分析重点
- **整体格局**：吉凶判断和总体趋势

### 分析与建议

根据问题类型给出针对性分析：

- **用神分析**：分析用神宫位的星、门、神组合
- **吉凶判断**：综合判断整体吉凶
- **行动建议**：给出具体的行动指导
- **时间建议**：根据时辰给出时机建议
- **注意事项**：提示需要防范的问题

## 项目结构

```
qimen-tool/
├── index.html              # 主页面
├── server.js               # Express 服务器
├── package.json            # 项目配置
├── README.md               # 项目说明
├── css/
│   ├── style.css           # 主样式文件
│   └── animations.css      # 动画样式文件
├── js/
│   ├── qimen.js            # 奇门遁甲排盘算法
│   ├── symbols.js          # 符号详细数据
│   ├── ui.js               # 界面交互模块
│   ├── starry.js           # 星空背景动画
│   └── app.js              # 主应用模块
└── data/
    └── explanations.json   # 详细解释数据
```

### 模块说明

| 模块 | 功能 |
|------|------|
| `qimen.js` | 排盘算法：四柱干支计算、阴阳遁判断、局数计算、九宫排盘 |
| `symbols.js` | 符号数据：九星、八门、八神、九宫的详细解释 |
| `ui.js` | 界面交互：九宫格渲染、符号解读、元信息显示 |
| `starry.js` | 星空背景：星星生成、流星动画 |
| `app.js` | 主应用：初始化、事件绑定、盘象总结、分析建议 |

## 开发指南

### 添加新符号

在 `symbols.js` 中添加：

```javascript
{
  name: '符号名称',
  category: '分类',
  element: '五行',
  luck: '吉凶',
  meaning: '含义',
  explanation: '详细解释',
  examples: ['示例1', '示例2']
}
```

### 修改样式

- 主题颜色：修改 `css/style.css` 中的 CSS 变量
- 动画效果：修改 `css/animations.css` 中的关键帧
- 响应式断点：修改媒体查询中的宽度值

### 扩展功能

- 添加新的标签页：在 `app.js` 中添加新的内容生成函数
- 修改排盘算法：编辑 `qimen.js` 中的计算函数
- 添加问题类型：在 `app.js` 的 `getShenPalace` 函数中添加新的关键词

## 注意事项

### 使用限制

- 本工具仅供教学和研究使用
- 排盘结果仅供参考，不作为决策依据
- 请勿用于商业用途或迷信活动

### 技术说明

- 排盘算法基于传统奇门遁甲理论，进行了适当简化
- 阴阳遁判断采用简化方法（1-9月阳遁，10-12月阴遁）
- 局数计算采用简化公式，实际应用中可能需要更复杂的节气计算

### 浏览器兼容性

- 支持现代浏览器（Chrome、Firefox、Safari、Edge）
- 建议使用最新版本浏览器以获得最佳体验
